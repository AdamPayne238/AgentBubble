# ChatGPT Setup

Use AgentBubble with ChatGPT by keeping `.agentbubble/` as the source of truth.

ChatGPT may not automatically read local project files unless connected through a tool or environment.

When needed:

- paste the relevant `.agentbubble/` files
- upload the relevant `.agentbubble/` files
- provide direct access through the connected environment

Avoid pasting the full framework unless necessary.

Use `prompts/start-session.md` as the normal bootstrap.
