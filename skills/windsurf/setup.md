# Windsurf Setup

Use AgentBubble by keeping `.agentbubble/` at the project root.

`.agentbubble/` remains the source of truth.

Windsurf rules can point to:

- `.agentbubble/session-start.md`
- `.agentbubble/coding-rules.md`

Do not duplicate the whole framework into Windsurf rules.

Use rules as lightweight pointers.
